# Artículo 0 — Resumen del Dongle Soberano

La existencia y competencia de un único custodio humano  
son la base de toda autoridad del Commonwealth.

Responsabilidades:
- Estar vivo y capaz de firmar  
- Aprobación/veto final de cambios y despliegues  
- Soportar deriva ética/psicológica/política no delegable  
- Si el Dongle falla, toda expansión autónoma se detiene

Forma mínima:
> Mantener el cuerpo vivo para ejercer autoridad de firma.
